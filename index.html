Operadores y Estructuras de control (Ciclos).

Operadores

Son símbolos o palabras clave que realizan operaciones en variables y valores. Pueden realizar tareas como la adición, la resta, la multiplicación, la comparación, la asignación, entre otras.

Operadores Comparación:

  • ++ : Incremento
  • - : Decremento

Operadores Aritméticos:

  • + : Suma
  • : Resta
  • : Multiplicación
  • / : División
  • % : Módulo (resto de la división)

Operadores Asignación:

  • = : Asignación
  • += : Suma y asignación
  • = : Resta y asignación
  • = : Multiplicación y asignación
  • /= : División y asignación

Operadores Comparación:

  • == : Igual a
  • != o <> : No igual a
  • < : Menor que
  • > : Mayor que
  • <= : Menor o igual que
  • >= : Mayor o igual que

Operadores Comparación:

  • && o and : AND lógico
  • || o or : OR lógico
  • ! o not : NOT lógico

Operadores de asignación:

En PHP tambien se utiliza para sumar, restar, multiplicar o dividir el valor de la derecha al valor de la izquierda y luego asignar el resultado a la variable de la izquierda. Es una forma abreviada de escribir $variable = $variable + valor.

// Ejemplo practico.
<?php
	$numero = 5;
	$numero += 3; // Equivalente a $numero = $numero + 3;
	
	echo $numero; // Imprimirá en pantalla el resultado = 8
?>

Definicion de que es un ciclo.

En general, un bucle es una estructura de control que permite repetir un bloque de instrucciones las veces que sea necesario. El bloque de instrucciones que se repite se suele llamar cuerpo del bucle y cada repetición se suele llamar iteración. PHP dispone de varias instrucciones para definir bucles.

Ciclo for

Es una estructura de control que permite ejecutar un bloque de código varias veces, de acuerdo con una condición específica.

// La sintaxis básica de un bucle "for" en PHP es la siguiente:
for (inicialización; condición; incremento/decremento) {
    // código a ejecutar en cada iteración
}
// Ejemplo practico con ciclo for
for ($i = 1; $i <= 5; $i++) {
    echo $i . " ";
}
// Ejemplo practico
// Suma de los primeros 10 digitos usando ciclo for y operadores de asignación.
<?php
// Declaramos la variable y la inicializamos en 0
	$suma = 0;
	
	// Definimos una estructura de control que se va a repetir 10 veces
	// iniciando del "1". e incrementandose de "1 en 1".
	for ($i = 1; $i <= 10; $i++) {
			// Realizamos la suma
	    $suma += $i;
	}
	// Imprimimos en pantalla el resultado.
	echo "La suma de los primeros 10 números es: " . $suma;
	?>

Ciclo While

El ciclo while se utiliza para repetir un bloque de código mientras una condición especificada sea verdadera.

// La sintaxis básica del ciclo while es la siguiente:
while (condición) {
    // código a ejecutar mientras la condición sea verdadera
}
// EJEMPLO PRACTICO USANDO EL CICLO WHILE:
// Desplegar 5 numeros en pantalla.

// Declaramos la variable y la inicializamos en 1.
$contador = 1;

// Definimos la estructura de control en donde la condicion es:
// Si la variable es menor o igual que 5, despliegame en pantalla.
while ($contador <= 5) {
		// Hacemos una impresion en pantalla
    echo "Iteración $contador <br>";
		// Incrementamos de "1 en 1"
    $contador++;
}
// OUPUT
Iteración 1
Iteración 2
Iteración 3
Iteración 4
Iteración 5

Ciclo Do While

Bucle que ejecuta un bloque de código mientras una condición especificada sea verdadera. La diferencia principal entre do-while y while es que do-while asegura que el bloque de código se ejecute al menos una vez, ya que la condición se evalúa después de la ejecución del bloque.

// La sintaxis básica de un bucle do-while en PHP es la siguiente:
do {
    // código a ejecutar
} while (condición);
// EJEMPLO PRACTICO USANDO EL CICLO DO WHILE:
// Desplegar 5 numeros en pantalla.
<?php
	$numero = 1;
	
	do {
	    echo $numero . " ";
	    $numero++;
	} while ($numero <= 5);
?>
// OUTPUT
1 2 3 4 5
